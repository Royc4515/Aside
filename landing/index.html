Why Aside

Built to disappear into your workflow.

You're reading something — an article, a paper, an API doc — and you want an AI's take, right now, without leaving the page.

Reads the page for you

Streaming answers

How it works

Three steps to a smarter page.

  1. 1

    Press Alt+A

    A slim sidebar slides in on the right (or left — your choice). It already knows the page's language and what text you have selected.

  2. 2

    Click a chip — or type

    Summarize, Key points, Translate, Explain, Find on page. Or write your own prompt. Answers stream in token-by-token.

  3. 3

    Switch models any time

    The header dropdown swaps between Claude, Gemini, GPT-4o, Grok, Groq, and Ollama with one click. Your active provider persists between sessions.

Privacy

Your keys. Your machine. Your data.

Keys stay local

API keys live in Chrome's encrypted storage and are sent only to the provider you chose, over HTTPS.

No middleman

No analytics, no telemetry, no proxy server. Requests go straight from your browser to the model provider.

Only on your prompt

Page content is included only when you ask. Trimmed to 12,000 characters and sent with that single request.

Fully offline mode

Pick Ollama and nothing leaves your computer — your prompt and the page text stay on your machine.

FAQ

Common questions.

Is Aside really free?
Yes. Aside itself is open source under the MIT license, free to use. You pay only what your provider charges for API calls — Anthropic, Google, OpenAI, xAI, and Groq each set their own pricing. Pick Ollama and run it locally — the whole stack is free.
Are my API keys safe?
Keys live in Chrome's encrypted storage and are sent only to the provider that issued them, over HTTPS. There is no Aside server, no proxy, no analytics, no telemetry. Read the full Privacy Policy.
Do I need an account to use Aside?
No account, no sign-up, no email. Install the extension, paste at least one provider API key, and you're in. Aside has no login of its own.
Which browsers are supported?
Aside is a Manifest V3 extension and runs on any Chromium-based browser — Google Chrome, Microsoft Edge, Brave, Arc, Opera. Firefox is on the roadmap — we'll add it once we port the manifest.
Can I use Aside fully offline?
Yes. Pick Ollama as the provider and Aside makes no network calls outside localhost. Your prompts, selected text, and page content stay on your machine.
Where is my conversation history stored?
Threads are saved per-site in Chrome's storage.local on your device. They aren't synced to any cloud and can be cleared from Settings or by uninstalling the extension.
Alt+A doesn't open the sidebar — how do I fix it?
Chrome doesn't auto-assign keyboard shortcuts to unpacked extensions (only to Chrome Web Store installs). Open chrome://extensions/shortcuts, find Aside, and set Toggle the AI sidebar to Alt+A (or any combo you like). 10 seconds.
Install

Two ways to install. A minute either way.

Chrome Web Store

Coming soon

One-click install, automatic updates. We're preparing the store listing — check back shortly.

Load unpacked

Available now
  1. Download the repo — latest ZIP — and unzip it.
  2. Open chrome://extensions and turn on Developer mode.
  3. Click Load unpacked and pick the folder that contains manifest.json (the Aside-main folder you just unzipped — not a subfolder).
  4. Open Settings and paste at least one API key.
  5. Hit Alt+A on any page.
  6. If the shortcut doesn't respond, open chrome://extensions/shortcuts and set Alt+A for Toggle the AI sidebar — Chrome doesn't always auto-assign keys to unpacked extensions.
