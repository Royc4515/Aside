Support

Need help, found a bug, or want to suggest a feature?

Open an issue

The fastest way to get help is to open an issue on GitHub. Bug reports, feature requests, and questions are all welcome.

Open a GitHub issue

Before reporting a bug

  • Reload the extension from chrome://extensions — many issues disappear on reload.
  • Check the browser console (right-click the sidebar → Inspect) and include any error messages.
  • Note your Chrome version and the provider you were using.

API key issues

  • Aside never stores keys on any server — there is no server. If a key looks wrong, paste it again in the Options page.
  • Each provider has its own usage dashboard. If requests fail with 401/403, check the key on the provider’s side first.
  • Rate-limit errors (429) come from the provider, not from Aside.
